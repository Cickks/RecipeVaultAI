# Design System

## Personality
Warm, calm, premium, fast, and organized. RecipeVault AI should feel closer to Pinterest plus Apple Reminders than a spreadsheet.

## Colors
- Cream: `#FFF8EF`
- Paper: `#FFFFFF`
- Ink: `#241F1C`
- Cocoa: `#6B4F3F`
- Sage: `#8FAE8A`
- Tomato: `#D9644A`
- Honey: `#F4B860`
- Mist: `#F3EFE8`
- Dark ink: `#151312`

## Typography
- Use platform system fonts initially.
- Large editorial titles for recipe and dashboard moments.
- Compact, readable labels for metadata, tags, and filters.

## Components
- Recipe card
- Ingredient card
- Step card
- Image gallery
- Bottom navigation
- Search bar
- Floating add button
- Profile header
- Collection card
- Meal planner card
- Shopping item
- Favorite button
- Tag chips
- Skeleton loaders

## Accessibility
- Minimum touch target: 44x44.
- Do not rely on color alone.
- Use accessibility labels for icon buttons.
- Support dark mode with automatic system preference.
